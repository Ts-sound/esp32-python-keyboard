# ESP32 Keyboard 系统设计

## 概述

ESP32 Python Keyboard 是一个基于 MicroPython 的 BLE HID 键盘项目，支持 WiFi 远程控制和自动按键功能。

## 数据流

### 消息队列流程

```
WiFi 接收 → msg_queue.publish() → 回调函数 → RF4 服务处理
```

### HID 报告流程

```
应用层 → keyboard_device.press() → hid_driver.send_keys() → BLE 广播
```

## 配置管理

所有配置参数位于 `config/config.py`：
- WiFi 配置（SSID、密码、端口）
- RF4 配置（按压时间、释放时间）
- HID 配置（设备名称、报告间隔）
- 硬件配置（LED 引脚）

## 依赖

- **MicroPythonBLEHID**: BLE HID 服务库 (`lib/hid_services.py`)
- **MicroPython**: v1.18+ (推荐 v1.23+)
